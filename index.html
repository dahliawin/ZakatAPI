Zakat API

Zakat API adalah web API untuk melakukan perhitungan zakat (zakat maal dan zakat penghasilan) serta memberikan informasi rumah zakat yang ada di sekitar kita.

Untuk menggunakan web API ini, pengguna harus melakukan registrasi dengan URL: zakatapi.org/register/user=USERNAME&password=PASSWORD

Untuk mendapatkan nilai zakat maal dapat menggunakan URL: zakatapi.org/zakat=maal&harta=TOTAL_HARTA_DALAM_SETAHUN&key=KEY.
*KEY diberikan saat registrasi

Untuk mendapatkan nilai zakat penghasilan dapat menggunakan URL: zakatapi.org/zakat=penghasilan&pendapatan=TOTAL_PENDAPATAN_BULANAN&pengeluaran=TOTAL_KEBUTUHAN_BULANAN&key=KEY.
*KEY diberikan saat registrasi

Untuk mendapatkan lokasi penyaluran zakat terdekat dapat menggunakan URL: zakatapi.org/lokasi/key=KEY.
*KEY diberikan saat registrasi
